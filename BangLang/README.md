## What is this?

**Bangalore Lang** is a Python wrapper that replaces boring keywords with peak Bangalore meme slang. It allows you to write code that sounds like a heated argument in Indiranagar.

Instead of `print`, you say `tell_da`. Instead of `if`, you say `yaavuh`.

It runs on top of Python, so you have access to all standard Python librariesâ€”just with better syntax.
