## What is this?

**Bangalore Lang** is a Python wrapper that replaces boring keywords with Bangalore meme slang.

Instead of `print`, you say `tell_da`. Instead of `if`, you say `yaavuh`.

It runs on top of Python, so you have access to all standard Python librariesâ€”just with better syntax.

Here is the translation map you need to know to survive in this codebase:

| Standard Python | Bangalore Lang     | Example Usage            |

| `def`           | **`put_scene`**    | `put_scene my_func():`   |
